## Phase 3 TailwindCSS Component Selection

For this phase, the dashboard will use the following TailwindCSS components:

- Header: Used to display the portfolio title.
- Navigation bar: Used to link to Home, Phase 1, Phase 2, Phase 3, Phase 4 and Phase 5.
- Dashboard cards: Used to display each portfolio phase as an individual card.
- Footer: Used to display copyright and basic site information.
- Copyright page: Used to list resources used in the site.

The selected components are based on the ScreenCraft component examples and have been adapted to use semantic HTML elements where appropriate.
